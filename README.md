## About
### 项目名 sahara(撒哈拉沙漠)
  本项目是一套管理系统, 开发初衷是由于本人专注后端, 对前端技术和框架了解不多, 所以开发一套可配置, 可修改, 自适应的后台管理系统,
正好也可以应用到个人博客, 充当管理系统.
  项目名即兴起的, 向往撒哈拉沙漠, 希望有机会去一次.

## 技术栈
- **nodejs**
- **vue**
- **bootstrap**

## 项目运行

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
